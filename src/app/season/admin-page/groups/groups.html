Approvers

Members of the approvers team can:
  • Change a request's status (approval, purchased, distributed)

Acquisitions

Members of the acquisitions team have the same permissions of the approvers team, plus they can:
  • Create/view reports
  • Edit any project request
  • Make extra request modifications, e.g. change purchaser, allocation, and cost adjustment
  • View inventory page
  • View allocations page

Admin

Members of the admin team have the same permissions of the acquisition team, plus they can:
  • View the admin page
  • Create new site messages
  • Modify group users
  • Create/modify events
  • Create/modify projects
